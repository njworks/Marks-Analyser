/*
*How to compile the Java Files:
*/
1) Download the JDK file: http://www.oracle.com/technetwork/java/javase/downloads/index.html 
2a) On Windows: Start -> Computer -> System Properties -> Advanced system settings -> 
   Environment Variables -> System variables -> PATH.
2b) On Apple: https://docs.oracle.com/javase/8/docs/technotes/guides/install/mac_jdk.html
3) From the Command Prompt to Compile: C:\Users\username>cd c:\group16\
                             	       C:\group16\> javac *.java
4) To execute the program: C:\group16\> java run.java


/*
*How to run the executable group16.jar file:
*/
1) Open the jar file
2) Select csv ExampleDataFileToBeConvertedIntoGraphs file
3) View the student analysis graphs and table
4) Export to PDF


/*
*Platform OS:
*/
1) Runs on both Windows and Mac Systems, with JDK 7 or higher
